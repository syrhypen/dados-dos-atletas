# dados-dos-atletas
Projeto de certificação 2 – Dados dos atletas
Aluno: Lucas Gabriel da Silva Cruz

*Sobre o projeto*
Este projeto foi desenvolvido em JS com o objetivo de representar atletas e calcular informações importantes como categoria, IMC e média válida das notas.

O sistema utiliza Programação Orientada a Objetos para organizar os dados e comportamentos de cada atleta.
